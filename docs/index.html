Documentation

For a detailed description of the improved calibration, see "Improved Spectrophotometric Calibration of the SDSS-III BOSS Quasar Sample" (Margala et al., 2015)

We provide corrections to all "offset" targets from the v5_7_0 processing of DR12. An "offset" target is identifiable via the LAMBDA_EFF keyword in spAll, spPlate, or individual spec files, see the data model for more details. For most BOSS targets (i.e. galaxies), LAMBDA_EFF has a value of 5400 A. For "offset" targets, LAMBDA_EFF has a value of 4000 A. We additionally require that the fiber was plugged (i.e. that the UNPLUGGGED bit of the ZWARNING bitmask is not set).

Downloading Corrections for DR12

The current version is tpcorr-0.1, available here.

The corrections can be unpacked using tar via the command: 

  
  $ tar xzf tpcorr-0.1.tar.gz

This will create a folder named tpcorr-0.1, which contains 3 files:
  • README.txt: A text file containing a link to this page.
  • example_usage.py: An example python program that makes use of the corrections.
  • tpcorr.hdf5: An HDF5 file that contains the corrections. The correction for each target is stored as a separate HDF5 Dataset (in python, this similar to a numpy array). The datasets are organized into HDF5 groups and subgroups by SDSS plate ID, observation MJD, and target fiber ID (in python, this is similar to nested dictionaries).
