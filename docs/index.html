Downloading Corrections for DR12

The current version is tpcorr-0.1, available here. Note, that this is currently a release candidate and not yet finalized.

The corrections can be unpacked using tar via the command: 

  
  $ tar xzf tpcorr-0.1.tar.gz

This will create a folder named tpcorr-0.1, which contains 3 files:
  • README.txt: A text file containing a link to this page.
  • example_usage.py: An example python program that makes use of the corrections.
  • tpcorr.hdf5: An HDF5 file that contains the corrections. The correction for each target is stored as a separate HDF5 Dataset (in python, this similar to a numpy array). The datasets are organized into HDF5 groups and subgroups by SDSS plate ID, observation MJD, and target fiber ID (in python, this is similar to nested dictionaries).
